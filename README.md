# 🎮 Quiz Game

A fun, interactive quiz game with 9 different categories!

## 📋 Categories
- 🌍 Geography
- 📜 History
- 🔬 Science
- ⚽ Football
- 🎬 Movies & TV Shows
- 🎮 Video Games
- ♟️ Chess
- ➗ Mathematics
- 💻 Programming

## 🎯 How to Play
1. Run `python quiz_game.py`
2. Choose a category (1-10)
3. Answer 10 questions
4. Get your score and feedback!

## 🚀 Features
- 90 unique questions (10 per category)
- Score tracking
- Motivational messages based on performance
- Colorful emojis for better UI

## 🛠️ Requirements
- Python 3.x
- No external libraries needed!
